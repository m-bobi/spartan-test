Spartan

What is Spartan?

You may be asking yourself "What is Spartan?" and I'm here to tell you exactly that! Spartan is a discord bot which has many features!
Spartan can do all sorts of things, ranging from Moderating, playing Music, getting Information on users in your server etc..
Spartan, also uses the Last.FM API from which you can scrobble your spotify plays, see how many people know a certain artist/song, and more!
Invite Spartan to check all these out!

.avatar - Shows the avatar of a mentioned user, or your own.
.dev - Get to know the developer!
.invites - Check how many people a user has invited.
.anime - Searches for an anime in Kitsu.io!
.manga - Searches for a manga in Kitsu.io!
.color - Shows a random color with a preview of it.
.edits - Fetch the recent edits of a message.
.enlarge - Makes an emote bigger, if possible.
.firstmessage - Get the first message on a channel
.invite - Invite Spartan!
.snipe - Fetch the last deleted message
.bugreport - Report any bugs you find within Spartan!
.urban - Searches your query on the Urban Dictionary.

.backup-create - Creates a backup for the server.
.backup-load - Loads a created backup.
.ban - Bans the given member's id or name.
.list-bans - DMs you a list of banned users.
.kick - Kicks a tagged member.
.lockdown - Prevents users from messaging in the current channel!
.mute - Mute the tagged user (if the admin has made the role "Muted")
.nuke - Clones the current channel, and deletes the channel where the command was ran.
.purge - Deletes up to 99 messages from the current channel.
.rule - Set up default rules for your server.
.unban - Unbans a user.
.unmute - Unmute tagged users.
